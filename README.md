                                       Serverless Automation Pipeline (AWS)

Project Overview
    This project automatically processes files uploaded to AWS S3 and sends notifications using AWS services.
removes manual work,savetime and reduce human error.concept i learn  Event-driven architecture,serverless computin,cloud automation,Backend workflow design

Problem Statement
   manual file tracking and notification is time-consuming.this project automates file logging and alerts using aws serverless services

 AWS Services Used
1.Amazon S3 (File Storage)
2.AWS Lambda (Backend Processing)
3.DynamoDB (Database)
4.SNS (Email Notification)

 How it works
1. File uploaded to S3
2. S3 triggers Lambda
3. Lambda stores data in DynamoDB
4. SNS sends email notification
